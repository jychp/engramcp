# Support

## How to get help

Use these channels depending on your need:

- Bug reports and feature requests: [GitHub Issues](https://github.com/jychp/engramcp/issues)
- Security reports: see `SECURITY.md` (do not open public issues)
- Maintainer contact: jeremy@subimage.io

## Before opening an issue

Please include:

- Expected behavior and actual behavior
- Reproduction steps
- Environment details (OS, Python version, package version)
- Logs or stack traces if relevant

## Scope of support

This project is maintained on a best-effort basis. Response times can vary based on maintainer availability and issue severity.
